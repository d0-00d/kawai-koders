# React + Express Skeleton App

This repository contains a minimal skeleton for a frontend React application and a backend Express API. The goal is to provide a clean structure for review and future implementation.

## Folder Structure

- `backend/` - Express backend with placeholder routes and middleware
- `frontend/` - React frontend with routing, tabs, and CSS Modules

## Run the Project with npm

1. Open a terminal in the repository root.
2. Install dependencies for both apps: `npm install`
3. Start the frontend app: `npm run start`
4. Start the backend app in a separate terminal: `npm run start:backend`

## Individual App Commands

- Frontend only: `npm run start:frontend`
- Backend only: `npm run start:backend`
- Build frontend: `npm run build:frontend`
